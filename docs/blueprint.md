# **App Name**: Logonova Agency

## Core Features:

- Homepage Display: Display key agency information including services and showcasing selected projects to engage potential clients. Should use large modern photography to showcase the brand.
- Service Details: Detail the services offered by Logonova, highlighting benefits, use cases and pricing to allow customers to explore each service.
- Interactive Portfolio: Allow filtering and exploration of projects to provide examples of past work.
- Project Details: Display a single case study that showcases a particular project, along with metrics.
- Contact Form: A contact form to capture inbound lead requests.

## Style Guidelines:

- Primary color: Bright Blue (#4285F4) to reflect a tech-forward and trustworthy image.
- Background color: Dark slate (#1E293B) provides a modern, tech-focused feel and ensures readability.
- Accent color: Light Blue (#7FADFF) for interactive elements, creating contrast against the dark background.
- Headline font: 'Space Grotesk', a geometric sans-serif, to give a modern and techy feel, appropriate for a forward thinking agency. Body font: 'Inter' for longer texts, providing optimal readability.
- A grid-based responsive layout with a sticky, responsive Navbar and a clear, structured Footer to maintain consistency across all devices.
- Use simple, outline-style icons from Lucide-react, consistently sized and colored to complement the minimalist aesthetic and improve user navigation.
